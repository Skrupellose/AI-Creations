## 提示词方向

- **真实快照感**：强调非专业构图、轻微运动模糊、JPEG 压缩痕迹、前景遮挡和自然互动感。
- **写实转换**：将动漫插画转换为真实照片，同时尽量保留原图构图、服装、姿势和环境。
- **富士胶片风格**：使用 Pro 400H / Superia 类似的柔和色调、低对比度、胶片颗粒、柔和高光和自然光。

## 使用方式

1. 打开 `images` 目录中的 `.md` 文件，复制或参考其中的提示词。
2. 在支持图像生成或图像编辑的 AI 工具中使用提示词。
3. 根据模型输出结果微调场景、光线、构图、角色细节和风格描述。
4. 将新的效果图保存到 `images` 目录，并补充对应的提示词文档。

## 维护建议

- 每组作品建议保持「提示词 Markdown + 效果图 PNG」的配对命名。
- 新增作品时，推荐使用序号开头，便于排序和索引。
- 如果提示词依赖参考图，建议在 Markdown 中标注参考图来源、目标比例和关键保留项。
